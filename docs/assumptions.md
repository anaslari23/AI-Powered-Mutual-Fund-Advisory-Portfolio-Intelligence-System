# Financial Assumptions

To compute financial models purely purely based on public data estimates:

- **Pre-Retirement Inflation**: Assumed at 6%
- **Education Inflation**: Assumed at 8%
- **Base Expected Return (Equity-heavy)**: Assumed at 12%
- **Monte Carlo Volatility**: Assumed 15% standard deviation for base portfolios.
- **Retirement Age**: Hardcoded rule-of-thumb age of 60.
